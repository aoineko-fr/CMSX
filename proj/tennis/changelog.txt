T:	Type of pattern (2-bits)
	0: N blocks of 0x00
	1: N blocks of the following byte value
	2: N blocks of the following 2 bytes values
	3: N blocks of uncompressed data
N:	Number of repetition of the blocks (6-bits: 1-64)
